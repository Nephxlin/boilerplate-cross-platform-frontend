# Atoms

**O que significa:** Os átomos são os menores blocos de construção da interface. Elementos HTML básicos estilizados, sem lógica de negócio, reutilizáveis em qualquer contexto.

**Tipos de componentes que pertencem aqui:**

- **Buttons** — Botões (primary, secondary, outline, ghost, etc.)
- **Inputs** — Campos de texto, email, password, etc.
- **Labels** — Rótulos de formulário
- **Icons** — Ícones isolados
- **Badges** — Etiquetas, tags, chips
- **Avatars** — Imagens de perfil circulares
- **Spinners / Loaders** — Indicadores de carregamento
- **Dividers** — Separadores visuais
- **Links** — Links de texto estilizados

Cada átomo deve ser **autocontido**, **sem dependência de outros componentes da _UI** e **documentado com Storybook**.
